About This Project

This assignment is a startup case-study dashboard inspired by research-style websites. It focuses on African startup failures and uses short, factual examples from selected countries.

The goal is to show how different sectors can fail for different reasons such as funding pressure, weak unit economics, operational problems, regulation, or lack of product-market fit.

The project uses seven categories as per now: Fintech, Ecommerce, Health & Biotech, Mobility & Deeptech, Web3 & Blockchain,Service providers and Social Media. Each category page contains a few verified case studies with the country and a short note about why the startup collapsed or shut down.

This website as per now is a personal class project dashboard to be reviewed and marked, and inspired by startup research sites and others related of the same theme but dont information of African startup failures as a whole.
